## 4. Analysis
The email showed signs of phishing due to suspicious sender details, unusual links or attachments, and social engineering tactics designed to pressure the user into taking action.

## 5. Risk
The main risks associated with this email include:
- Credential theft
- Malware infection
- Account compromise
- Financial fraud
- Data exposure

## 6. Recommended Remediation
- Report the email to the security team.
- Do not click links or open attachments.
- Block malicious sender/domain if confirmed.
- Reset the user's password if credentials were entered.
- Review MFA status.
- Scan the endpoint if an attachment was opened.
- Add IoCs to monitoring or block lists.
- Educate the affected user.

## 7. Lessons Learned
This project helped build practical understanding of phishing indicators, safe handling of IoCs, and how SOC analysts document suspicious email investigations.
