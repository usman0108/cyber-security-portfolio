# Phishing Email Analysis Project

## Project Overview
This project documents a phishing email analysis investigation completed as part of my cyber security portfolio development. The aim was to practise identifying suspicious email indicators, social engineering tactics, malicious links, attachments and indicators of compromise.

## Skills Practised
- Phishing email analysis
- Suspicious sender identification
- URL and attachment review
- Social engineering recognition
- Indicator of Compromise identification
- Risk assessment
- SOC-style reporting
- Remediation recommendations

## Key Findings
- Reviewed suspicious sender details and email content.
- Identified potential phishing indicators such as suspicious links, attachments and social engineering tactics.
- Practised defanging indicators such as IP addresses and URLs.
- Documented risk and recommended response actions.

## Recommended Response
- Report the email to the security team.
- Do not click suspicious links or open unknown attachments.
- Block malicious senders, domains, URLs or IP addresses where appropriate.
- Reset credentials if the user interacted with the phishing email.
- Review MFA status.
- Scan the endpoint if an attachment was opened.
- Add indicators to monitoring or block lists.
