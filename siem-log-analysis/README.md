# SIEM Log Analysis Project

## Project Overview
This project documents a beginner SIEM-style log analysis investigation using simulated web server logs. The aim was to practise identifying suspicious activity, reviewing indicators, building simple search logic, and documenting findings in a SOC-style report.

## Scenario
A simulated web server generated multiple log events showing suspicious activity from external IP addresses. The activity included repeated failed login attempts, suspicious URL paths, SQL injection-style requests, and possible reconnaissance behaviour.

## Skills Practised
- Log analysis
- SIEM-style investigation
- Suspicious IP identification
- Web attack indicator review
- Brute-force activity recognition
- SQL injection indicator recognition
- Incident documentation
- Remediation recommendations
