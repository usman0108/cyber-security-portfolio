# SOC Alert Investigation Project

## Project Overview
This project documents a beginner SOC alert investigation based on suspicious authentication activity. The aim was to practise reviewing login logs, identifying possible account compromise, and documenting recommended remediation steps in a SOC-style report.

## Scenario
A simulated alert was triggered after multiple failed login attempts were detected against a user account, followed by a successful login from the same suspicious source IP address. Additional activity included a password change and the creation of an external email forwarding rule.

## Skills Practised
- SOC alert triage
- Authentication log review
- Brute-force and credential attack identification
- Indicator of Compromise analysis
- Account compromise investigation
- Incident response recommendations
- SOC-style documentation
